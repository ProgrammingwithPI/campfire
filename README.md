Lab 453 

What is this project:
A web based pygame horror game taking place beneath the surface with underwater cyborgs.

Why does it exist:
Made for hack clubs campfire event, then refined for the ember game jam.

How I built it / why it works: (with help from AI)
Uses A* maze cyborgs navigation, and generates the maze using recursive backtracking for maze generation.
Gun drawing, shooting, eyes looking at the sub, squid moving its tentacles, distance finding all uses trig.
Used algebra for certain parts of the game.

Ai was used only for coding, normally I would create code and if it was buggy I would ask claude to fix it. Claude was also used to bring it to the web, and generating lab floor tile pattern, and the boss, along with into, but even in these functions I coded some of it like the eyes in lab intro, and the squids design in boss fight function.

For sounds and music:
Sfx and music weren't made by me I used pixabay for all the sounds, such as from dragon studios, I also used real sperm whale clicking sounds such as from NOAA.
